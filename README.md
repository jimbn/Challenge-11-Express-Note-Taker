# Challenge 11 - Express Note Taker

This application is a note taker that is created by using express.js.


## Installation

User must install node_modules.

## Usage

User can open localserver on PORT:3001 by inputting <node server.js>.

## Built with
* HTML
* CSS
* Node.js
* Express.js
